Как настроить на удалённом репозитории:
https://habr.com/ru/articles/755036/

Чтобы всё работало нормально, настраиваем(создаём если нет) файл:
пример:
~\.ssh\config
###### Start file #####
Host gitverse.ru
    HostName gitverse.ru
    IdentityFile ~/.ssh/id_rsa
    IdentitiesOnly yes
Host gitlab.com
    HostName gitlab.com
    IdentityFile ~/.ssh/id_rsa
    IdentitiesOnly yes
Host github.com
    HostName github.com
    IdentityFile ~/.ssh/id_rsa
    IdentitiesOnly yes
#####  End file  #####
