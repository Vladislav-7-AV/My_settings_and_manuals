Alt + стрелка вверх/стрелка вниз — двигает ячейку

--
Shift + Tab — сместить табуляцию на один шаг влево.
Shift + Del — удалить строку целиком
Ctrl + D — выделяет слово
Ctrl + L — выделяет всю строку

Быстро добавить комментарий
Windows — Ctrl + /
macOS — Command + /

Перейти к строке под номером
Windows — Ctrl + G
macOS — Control + G

Поменять строку местами с соседними
Windows — Alt + ↑ / ↓
macOS  Option + ↑ / ↓

Дублировать строку
Windows  Shift + Alt + ↓ / ↑
macOS  Shift + Option + ↓ / ↑


Перейти к парной скобке
Windows  Ctrl + Shift + \
macOS  Shift + Command + \

Переименовать переменную
Windows — F2
macOS — F2

Отформатировать документ
Windows — Shift + Alt + F
macOS  Shift + Option + F

Перейти к переменной
Windows — F12
macOS — F12

Включить или выключить перенос слов
Windows — Alt + Z
macOS  Option + Z

Включить дзен-режим
Windows — Ctrl + KZ
macOS — Command + KZ
Лучшее решение для тех, кто входит в состояние потока, когда пишет код. Дзен-режим скрывает все панели, разворачивает редактор на весь экран, и вы можете плыть в волнах кода и думать только о нём. Чтобы вернуться в реальный мир, нажмите Escape.

Ctrl + Alt + → — разделить рабочую область и переместить актуальную вкладку вправо. Ctrl + Alt + ← возвращает вкладку назад. Вы не поверите, насколько удобно видеть, например, models.py и views.py рядом.
